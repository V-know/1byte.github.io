Deploy Graphite with Grafana on Ubuntu 14.04 (Part Ⅱ)

Monitoring
  • Monitoring
  • grafana
  • graphite
  • ubuntu

Grafana 是 Graphite仪表盘和图形编辑器。Grafana 是开源的、功能齐全的度量仪表盘和图形编辑器，支持 Graphite，InfluxDB 和 OpenTSDB。

Grafana 主要特性：

  • 灵活丰富的图形化选项；
  • 可以混合多种风格；
  • 支持白天和夜间模式；
  • 多个数据源；
  • Graphite 和 InfluxDB 查询编辑器等等。
